ABOUT HOOK PIECE RATE CALCULATOR

This web application was developed for the Chennai Port Authority to streamline and automate the calculation of hook-based piece rate incentives for dock workers.

Key Features
  • Supports both Prediction and Planning calculation modes
  • Handles up to 4 hooks simultaneously
  • Automatically calculates piece rate earnings based on tons handled
  • Includes employee incentive breakdowns by role
  • Tracks absenteeism and adjusts calculations accordingly
  • Separates on-board and on-shore earnings
  • Export functionality to Excel for record keeping
Technical Details

The application uses modern web technologies:

  • Frontend: HTML5, CSS3, JavaScript (ES6+)
  • Data: JSON for hook line configurations
  • Export: SheetJS library for Excel generation
  • Responsive Design: Works on desktop and mobile devices
Calculation Methodology

The system implements the official Chennai Port Authority piece rate formulas:

  • Calculates PR Earning Tons (Piece Rate Tons - Datum Per Crane)
  • Applies different rates for productivity bands (100-150%, 150-200%, 200%+)
  • Distributes earnings across employee roles (Tindal, Maistry, Winch drivers, etc.)
  • Adjusts for absentees while maintaining total payout amounts
Usage Guidelines
  • Select date and enter vessel details
  • Choose between Prediction or Planning mode
  • Enter the number of hooks (1-4)
  • Fill in the required fields for each hook
  • Use the calculation buttons to generate results
  • Export to Excel when complete
