ABOUT THIS APPLICATION

This web application was developed for the Chennai Port Authority to streamline and automate the calculation of hook-based piece rate incentives for dock workers. It enables accurate and efficient handling of planning and prediction operations by incorporating real-time input fields and structured rate band calculations.

Key Features
  • Responsive, user-friendly interface optimized for desktop and mobile.
  • Supports both Prediction and Planning modes of calculation.
  • Auto-fetch and compute values based on line data from a master dataset.
  • Breakdown of earnings by incentive bands (101–150%, 151–200%, Above 200%).
  • Export-ready results display for recordkeeping or reporting.
Technology Stack
  • Frontend: HTML, CSS (custom styles), and vanilla JavaScript.
  • Fonts & Icons: Google Fonts (Roboto).
  • Data Handling: JSON for dynamic dropdown and calculation reference.
Project Purpose

The goal of this project is to enhance operational transparency, reduce manual effort, and ensure accurate wage calculations in alignment with port regulations and labor agreements.
